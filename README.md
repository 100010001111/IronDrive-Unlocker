🔓 IronDrive Unlocker 🔓

O IronDrive Unlocker é uma ferramenta de automação desenvolvida para facilitar a recuperação de acesso a volumes criptografados pelo BitLocker através de dicionário (wordlist). O projeto usa scripts Batch e PowerShell para interagir diretamente com as APIs de segurança do Windows.

Todos os direitos reservados. Não é permitido o uso, cópia ou comercialização deste código sem autorização prévia

🛠️ Tecnologias Utilizadas

Batch (CMD): Script de inicialização com elevação automática de privilégios (UAC).

PowerShell: Motor principal para processamento de strings seguras e comando Unlock-BitLocker.

🚀 Funcionalidades:


Primeiramente, ao executar o script .BAT, automaticamente solicita permissão de administrador e exige permissão do usuário

Utiliza uma wordlist com centenas de senhas 48 dígitos (Não chaves de recuperação)

Verificação de disco: Monitora se o SSD/Drive está conectado para a execução, assim consegue evitar erros de "vácuo"
